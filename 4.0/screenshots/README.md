# Screenshots Instructions

## Required Screenshots for GitHub Repository

Please capture and save the following screenshots in this folder:

### 1. **01_login.png**
- Navigate to http://127.0.0.1:5000
- Capture the login page showing:
  - Username and password fields
  - Login button
  - Register New Account button

### 2. **02_registration.png**
- Click "Register New Account" button
- Capture the registration form showing:
  - Full Name field
  - Username field
  - Password field
  - Register button
  - Back to Login button

### 3. **03_dashboard.png**
- After successful login
- Capture the dashboard showing:
  - Welcome message with user name
  - "Enter Exam Data" button
  - "Logout" button

### 4. **04_exam_entry.png**
- Click "Enter Exam Data"
- Fill in all exam details
- Generate student fields
- Capture the form showing:
  - Department, Year, Internal Assessment, Subject, Date fields
  - Number of Students field
  - Generated student input fields (Roll No, Name, Marks)
  - Submit & Analyze button

### 5. **05_analysis.png**
- After submitting exam data
- Capture the analysis dashboard showing:
  - Statistics cards (Total, Present, Absent, Highest, Lowest, Average)
  - Ranked student performance table with color coding:
    - Green rows for high performers
    - Yellow rows for average performers
    - Red rows for students needing attention
  - Back to Dashboard button

### 6. **06_data_files.png** (Optional)
- Open users.json and records.json in a text editor
- Capture showing the JSON data structure
- This proves data persistence

## How to Capture Screenshots

### Windows:
- Press `Windows + Shift + S` for Snipping Tool
- Or use `PrtScn` key and paste in Paint

### Mac:
- Press `Cmd + Shift + 4` for selection screenshot
- Or `Cmd + Shift + 3` for full screen

### Linux:
- Use Screenshot tool or `gnome-screenshot`

## Screenshot Specifications

- **Format**: PNG (preferred) or JPG
- **Resolution**: Minimum 1280x720
- **Quality**: High quality, readable text
- **Browser**: Use Chrome or Firefox for consistency
- **Zoom**: 100% browser zoom level

## After Capturing

1. Save all screenshots in this `screenshots/` folder
2. Name them exactly as specified above
3. Verify all text is readable
4. Ensure color coding is visible in analysis screenshot
5. Commit and push to GitHub
